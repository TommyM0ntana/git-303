Basic usage of GitHub

Simple Comands

1. MKDIR 'name of the folder'(Shortcut to Create a Folder)

2. GIT INIT (track changes inside the folder)

3. TOUCH index.html (create HTML file)

4. GIT STATUS (shows you what has been changed until last time)
5. COMMIT (Snapshot of your file, it useful when you want to go back if you have a problem)
